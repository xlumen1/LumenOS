*Open Source Hobby Operating System, What More Would You Want?*

## Dependencies
- gcc
- binutils
- grub-pc-bin
- xorriso
- qemu-system-x86
- mtools
- gdb

## To Build
```bash
make
```

Building And Running
```bash
make run
```

Building ISO
```bash
make iso
```

Launch With GDB
```bash
make debug
```

Force Recompile
```bash
make refresh
```
