Open Source Hobby Operating System, What More Would You Want?

## Dependencies
- gcc
- binutils
- grub-pc-bin
- xorriso
- qemu-system-x86
- mtools

## To Build
Just Building kernel.bin
```bash
make
```

Building And Running
```bash
make run
```

Building ISO
```bash
make iso
```
